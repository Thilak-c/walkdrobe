# 3D Models for Walkdrobe

Place your 3D shoe model here.

## Required File
- `shoe.glb` - Main hero shoe model

## Supported Formats
- `.glb` (recommended - binary, smaller file size)
- `.gltf` (JSON format with separate textures)

## Where to Get 3D Shoe Models

### Free Options:
1. **Sketchfab** - https://sketchfab.com/search?q=shoe&type=models&licenses=322a749bcfa841b29dff1571a5c83f2d
2. **Turbosquid** (free section) - https://www.turbosquid.com/Search/3D-Models/free/shoe
3. **CGTrader** (free section) - https://www.cgtrader.com/free-3d-models/shoe
4. **Free3D** - https://free3d.com/3d-models/shoe

### Tips:
- Look for models under 5MB for fast loading
- GLB format is preferred
- Make sure the model has good textures/materials
- Check the license allows commercial use

## How to Convert to GLB
If you have .obj, .fbx, or other formats:
1. Use Blender (free) - Import > Export as GLB
2. Online converter: https://products.aspose.app/3d/conversion/obj-to-glb

## Model Settings
The viewer expects:
- Model centered at origin
- Y-up orientation
- Reasonable scale (will be adjusted in code)
